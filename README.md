VMS Techs | Executive Real-Time Dashboard 📊
🚀 Project Overview
This project is a high-performance, real-time business intelligence tool designed for VMS Techs. It solves the critical problem of "Delayed Data Insights" by providing a live-streaming dashboard that visualizes Revenue, User Growth, and Risk Levels using JavaScript and API integration.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox & Grid)
Logic: JavaScript (ES6+)
Charts: Chart.js (Real-time streaming)
Icons: Font Awesome
Data Source: JSONPlaceholder API (Mocking real-time business streams)
💡 Key Features
Live Data Fetching: Uses the fetch API to simulate a constant data stream.
Dynamic Risk Assessment: Logic-based UI that changes status from "Low" to "High" based on revenue fluctuations.
Responsive Sidebar: Professional corporate navigation layout.
Zero-Refresh UI: The dashboard updates every 4 seconds without reloading the page.
